# TodoList ReactJS + Bootstrap

## Deskripsi
Aplikasi TodoList berbasis **ReactJS** dan **Bootstrap 5** (lokal).

---

## Fitur Utama

- **CRUD Tugas:** Tambah, edit, dan hapus tugas
- **Filter Tugas:** Berdasarkan prioritas (High/Medium/Low) dan status (To Do, In Progress, Done)
- **Pencarian:** Cari tugas berdasarkan judul atau deskripsi
- **Tanggal Deadline:** Setiap tugas bisa diberi batas waktu
- **Drag-and-Drop:** Urutkan tugas dengan drag & drop (dnd-kit)
- **Theme Switcher:** Ubah tampilan dark mode & light mode
- **Responsif:** Desain fleksibel untuk desktop dan mobile
- **Animasi:** Efek transisi pada kartu tugas (fade-in)

---

## Cara Instalasi & Penggunaan

1. **Clone repository:**
    ```
    git clone <YOUR-GITHUB-LINK>
    cd react-todolist-crud
    ```

2. **Install dependencies:**
    ```
    npm install
    ```
    Dependencies utama:
    - react
    - @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities
    - Bootstrap 5 lokal (link di public/index.html)

3. **Jalankan aplikasi:**
    ```
    npm start
    ```
    Buka [http://localhost:3000](http://localhost:3000) di browser.
